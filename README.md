# Bike Sharing Stations Data 🚲

## Overview
This project analyzes raw data from bike sharing stations mainly using Powerbi
It focuses on station details, usage patterns, and operational metrics to uncover insights into urban mobility and resource allocation.

---

## Dataset Columns
*(based on the uploaded file)*  
- Station ID  
- Station Name  
- Location (Latitude, Longitude)  
- Capacity (Number of docks)  
- Available Bikes  
- Available Docks  
- Timestamp / Date  

---

## Features
- Data cleaning and preparation of raw station records  
- Standardization of location and capacity fields  
- Time‑based analysis of bike availability  
- Identification of high‑demand and low‑demand stations  

---

## Outcomes
- **Cleaned Dataset**: Removed inconsistencies and standardized station information.  
- **Usage Insights**:  
  - Peak hours for bike demand  
  - Stations with frequent shortages or surpluses  
  - Correlation between station capacity and usage  
- **Operational Recommendations**:  
  - Redistribution strategies for balancing bikes across stations  
  - Capacity planning for high‑traffic areas  
- **Recruiter‑Ready Portfolio**: Demonstrates ability to handle raw transport data and generate actionable insights.
